# 🍕 Pizza Heaven - Modern Funky Pizza Shop Website

## Project Overview
A vibrant, funky pizza shop website with bold colors, playful animations, and memorable UI that captures the fun spirit of artisanal pizza.

## Visual Design Specification

### Color Palette
- **Primary**: `#FF4D4D` (Vibrant Red - tomatoes)
- **Secondary**: `#FFB347` (Warm Orange - cheese)
- **Accent**: `#FFE66D` (Sunny Yellow - fresh herbs)
- **Dark**: `#1A1A2E` (Deep navy - modern contrast)
- **Light**: `#FFF8F0` (Warm cream - background)
- **Text**: `#2D2D2D` (Charcoal)

### Typography
- **Headings**: "Fredoka One" - playful, rounded, bold
- **Body**: "Quicksand" - modern, friendly, readable
- **Accent**: "Permanent Marker" - for fun labels/tags

### Visual Style
- Bold, chunky borders and shadows
- Playful floating animations on pizza images
- Bouncy hover effects on buttons
- Wavy, organic shapes for sections
- Dotted patterns and geometric accents
- Hand-drawn style elements

## Component Breakdown

### 1. Navigation
- Floating nav bar with glassmorphism effect
- Animated logo (pizza emoji bouncing)
- Nav links with underline slide animation
- Mobile hamburger menu with fun transition

### 2. Hero Section
- Full-screen with animated pizza illustrations floating
- Bold headline with gradient text
- Fun tagline with typewriter effect
- Bouncy CTA button with hover animation
- Scattered decorative elements (pepperonis, leaves)

### 3. Menu Section
- Horizontal scrolling pizza cards
- Each card has:
  - Pizza illustration with hover zoom
  - Fun name with accent color
  - Price tag with pill shape
  - Ingredient tags
  - "Add to cart" button with ripple effect

### 4. About Section
- Split layout: image + content
- Story text with highlighted keywords
- Stats counters with animation (years, pizzas, happy customers)
- Wave separator at bottom

### 5. Footer
- Multi-column with brand info
- Social media icons with hover color changes
- Newsletter signup with fun input
- Copyright with pizza emoji animation

## Functionality

### Pages
1. **Home** - Hero, featured pizzas, about preview
2. **Menu** - Full pizza catalog with filtering
3. **About** - Brand story and team
4. **Contact** - Location, hours, contact form

### Interactions
- Smooth scroll between sections
- Parallax effects on scroll
- Cart sidebar with slide animation
- Form validation with fun feedback
- Loading states with pizza spinner

### Animations
- Floating pizza elements (CSS keyframes)
- Button bounce on hover
- Card scale on hover
- Staggered reveal on scroll
- Logo wiggle on load

## Responsive Breakpoints
- Mobile: < 768px (single column, stacked)
- Tablet: 768px - 1024px (2 columns)
- Desktop: > 1024px (full layout)
